- - - - Mini Social Post Application - - - -

This is a full-stack mini social media application built as part of the 3W Full Stack Internship Assignment.

Users can create an account, log in, create posts with text and/or images, view a public feed, like posts, and comment on them.

The project is inspired by the Social Page of the TaskPlanet app.

- - - -- Features - - - -

User Signup & Login (JWT Authentication)

Create Post with:

Text only

Image only

Text + Image

Public Feed (all users)

Like posts

Comment on posts

Logout functionality

Image upload handled using Multer

Responsive and clean UI
